In order to play: install the two zipped fonts, Open up the "Blocks" folder, open TileGame.sln, and press F5.

You can either play in story mode or creative mode. Prior to starting you can go into options and choose a character,
Or change sound/effects settings. 

Up, Left, Right to move your character, spacebar to create very simple AI. delete key to clear them.
Click any block to add/delete different types of blocks. Scroll to change block type. Pause with escape and save your scene.

Limitations:
You aren't allowed to create a new file with the same name as an existing one.
You cannot load a file that doesn't exist yet.
In story mode there is no editing allowed.
AI is for experimentation only and not game ready.

Experiment and have fun!:)
